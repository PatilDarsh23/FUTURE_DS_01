# Task 1: Business Sales Performance Analytics

## Objective
Analyze sales data to identify trends, top products, best categories, and regional performance.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Laptops generated the highest revenue among products.
- North and East regions performed better.
- Sales increased steadily from January to April.
- Electronics category dominates total revenue.
